# Privacy

Codex Obsidian is a local Codex plugin bundle. This repository does not add a hosted service, account system, analytics, or telemetry.

When a user runs bundled skills, Codex and local tools may read or write only the files and URLs the user asks them to handle. Obsidian CLI, Sync, Publish, and Defuddle behavior is governed by those tools and the user's own configuration.
